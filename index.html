Hypertext Markup Language (HTML) is the standard markup Language for creating web pages and web applications. Web browsers receive HTML documents from a web server or frim local storage and render them into multimedia web pages.HTML describes the structure of a web page semantically and orginally included cues for the appearance of the document.

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be create structed documents by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes and other items. HTML elements are delineated by tags, written using angele brackets.
